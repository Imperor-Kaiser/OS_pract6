System Information Utility
Консольная утилита на Bash для получения системной информации.

Функциональность
Утилита предоставляет следующие возможности:

Вывод пользователей (-u, --users):

Список всех пользователей системы
Их домашние директории
Отсортировано по алфавиту
Вывод процессов (-p, --processes):

Список запущенных процессов
Отсортировано по PID
Информация о пользователе и команде
Логирование (-l PATH, --log PATH):

Сохранение вывода в файл
Проверка прав доступа к файлу
Обработка ошибок (-e PATH, --errors PATH):

Сохранение сообщений stderr в файл
Фильтрация и обработка ошибок
Справка (-h, --help):

Подробное описание всех опций
Примеры использования
